# Redux — Редактирование + Фильтрация

[![Build Status](https://github.com/Pavel19740404/ahj-redux-editing/actions/workflows/deploy.yml/badge.svg)](https://github.com/Pavel19740404/ahj-redux-editing/actions/workflows/deploy.yml)

🔗 **[GitHub Pages — Live Demo](https://Pavel19740404.github.io/ahj-redux-editing/)**

CRUD-приложение на Redux Toolkit с редактированием и фильтрацией.

## Функциональность

- Добавление записей (название + цена)
- Редактирование: кнопка ✏️ заполняет форму, Save сохраняет, Cancel отменяет
- Удаление записей
- Фильтрация по названию (строковый поиск)

## Запуск

```bash
yarn install
yarn dev
```
